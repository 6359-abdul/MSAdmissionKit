Project Overview
The MSAdmissionKit is a digital portal for the MS Education Academy Admission Kit 2026. It serves as a comprehensive guide for prospective students and parents, covering everything from academic excellence to campus infrastructure.

How to Run Locally
To get the project running on your local machine for development, follow these steps:

Prerequisites
Node.js (Ensure you have the latest stable version installed)

Setup Instructions
Install dependencies:
Run the following command in your terminal to install the necessary packages:
npm install

Run the app:
Start the development server with:
npm run dev

Admission Kit Contents
The application features a 41-page digital kit, including:

Administrative Forms: Enquiry forms and office checklists.

Educational Philosophy: Mission, Vision, and the "5D Thinking Model."

Academic Highlights: Faculty profiles, IIT/NEET results, and the "MYJO" smart curriculum.

Student Life: Sports (Karate, Boxing, Judo), Science Olympiads, and Creative Arts.

Logistics: Fee structures and campus safety protocols.

Tip: You can navigate through the kit using the Vercel deployment to preview how the Markdown and assets render in the live environment.
